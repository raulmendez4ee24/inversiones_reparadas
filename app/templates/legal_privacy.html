Politica de privacidad

Como tratamos tu informacion cuando usas nuestro diagnostico y onboarding.

Inicio

1) Que datos recopilamos

  • Datos del negocio: giro/industria, region, descripcion de procesos, cuellos de botella, herramientas.
  • Datos de contacto (si los proporcionas): correo, canal preferido.
  • Preferencias: modulos seleccionados, tipo de bot, forma de entrega (WhatsApp/PDF/email).
  • Accesos tecnicos (solo si decides implementacion): tokens/API keys/IDs necesarios para conectar sistemas (ej. WhatsApp/Meta/CRM).
  • Datos tecnicos: logs minimos para diagnosticar errores (fecha/hora, endpoint, tipo de error).

Recomendacion: comparte accesos con usuarios tecnicos y permisos limitados; evita contrasenas personales.

2) Para que los usamos

  • Generar tu auditoria IA (fricciones, ROI estimado, modulos sugeridos y plan de trabajo).
  • Preparar la implementacion (conectar integraciones, validar tokens y armar flujos en n8n).
  • Darte acceso al portal para seguimiento (estatus, entregables y configuraciones).
  • Soporte y mejora del servicio (correccion de fallas y optimizacion del flujo).

3) Terceros y APIs

Para automatizar, a veces conectamos servicios de terceros (por ejemplo Meta/WhatsApp, CRMs, ERPs, email, n8n). Cada proveedor tiene sus propias politicas. Solo enviamos lo necesario para que el flujo funcione.

  • Meta/WhatsApp/Messenger: para mensajeria y webhooks (si lo solicitas).
  • Herramientas internas (n8n): orquestacion de flujos y automatizaciones.
  • LLM (opcional): si activas analisis con IA, parte del texto se envia al modelo para generar recomendaciones.

4) Almacenamiento y retencion

Guardamos la informacion en una base de datos para poder generar el reporte, habilitar el onboarding y dar seguimiento. Conservamos datos el tiempo necesario para operar el servicio y/o segun lo que se acuerde en la implementacion.

5) Seguridad

  • Recomendamos usar tokens revocables y permisos minimos.
  • El portal usa sesiones seguras; no compartas tu acceso con terceros.
  • Puedes solicitar eliminacion de datos en cualquier momento.

6) Derechos y contacto

Para dudas, correccion o eliminacion de datos escribe a {{ support_email }}. Incluye tu empresa y, si lo tienes, tu folio.
